Click the Buttons to Change the Map.

This is a quick demo of using an embedded Google Map api. The buttons on call a javascript method that changes src attribute in the iframe. The site is built with github.io pages using the page generator. To clone a copy of the project, open a terminal and enter
# git clone https://github.com/perezfr/GMapi
